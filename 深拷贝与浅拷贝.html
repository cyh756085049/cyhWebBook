深拷贝与浅拷贝

在 javascript 中有不同的方法来复制对象，那么我们怎样才能正确地复制一个对象呢？

浅拷贝

浅拷贝是创建一个新对象，这个对象有着原始对象属性值的一份精确拷贝。如果属性是基本类型，拷贝的就是基本类型的值，如果属性是引用类型，拷贝的就是内存地址 ，所以如果其中一个对象改变了这个地址，就会影响到另一个对象。

实现方式

1、通过Object.assign()实现

let a = {
  name: 'raymond'
};
let b = Object.assign({}, a);
a.name = 'ramona';
console.log(b.name);  // raymond

2、通过扩展运算符...实现

let a = { 
  name: 'raymond'
};
let b = {...a};
a.name = 'ramona';
console.log(b.name);  // raymond
